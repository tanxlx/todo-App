#  To-Do List App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript. It helps users manage their daily tasks efficiently.

##  Features
* Add new tasks quickly
* Mark tasks as completed
* Delete tasks when done
* Fully responsive design for mobile and desktop

## Tech Stack Used
* **HTML5:** For the structure of the application
* **CSS3:** For styling and beautiful UI
* **JavaScript (ES6):** For dynamic functionalities
